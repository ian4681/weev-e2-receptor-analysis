WEEV E2 V265F 受体向性切换的结构基础探索
基于 AlphaFold3 Server 的计算预测分析（两轮迭代）

📅 2026年5月 🔬 独立计算探索 📄 参考：Byrd et al., Cell Reports 2025 🛠 AlphaFold3 Server · Python · NumPy · Matplotlib

1. 研究背景与问题来源

2025年发表于 Cell Reports 的研究（Byrd et al.）报道了一个引人注目的发现： 西方马脑炎病毒（WEEV）E2 糖蛋白第265位的单点突变 V265F 足以改变病毒的受体向性—— 野生型（WT）利用 PCDH10 作为入胞受体，而 V265F 突变体则改用 VLDLR（低密度脂蛋白受体相关蛋白）。 这一"单点氨基酸切换受体"的现象意味着265位在结合界面上具有核心地位。

核心问题：V265（Val）→ F265（Phe）的侧链变化具体如何影响 E2 与 VLDLR / PCDH10 的结合界面？ 能否通过 AlphaFold3 在计算层面预测并比较两种构象的界面差异？

VLDLR 含有8个 LA（ligand-binding）重复结构域，文献已明确 LA3（UniProt P98155, pos 106–145） 是甲病毒结合的核心区域，关键残基为 P129 / W132 / D135 / E137 / D139。 PDB 中已有 WEEV E2 WT 的晶体结构（PDB 9L1N），但尚无 E2-VLDLR 复合物结构。 这正是 AF3 预测可以发挥作用的空间。

2. 实验设计：两轮迭代

本次分析分两轮提交，每轮解决不同问题：

R1
第一轮（全长受体，2026-05-23）
4个 job：WT / V265F × VLDLR全长（873aa）/ PCDH10胞外域（686aa）
目的 建立基线，验证 AF3 是否能区分两种受体
R2
第二轮（LA3截短，2026-05-23）
2个 job：WT / V265F × VLDLR LA2-LA4（pos 66–190，125aa）
动机 全长VLDLR有8个LA重复，AF3不知该对接哪个，导致信号被稀释； 截短至真实结合区（LA3±各一个重复）以提升信噪比

5. 结果解读

5.1 VLDLR 截短有效，但绝对置信度仍低

将 VLDLR 从873aa截至125aa后，WT 的界面 ipTM 从0.210升至0.290，pae_AB 从16.35降至13.33 Å。 这验证了"信号稀释"假说——全长VLDLR的8个LA重复使 AF3 无法聚焦真实结合区。 然而，0.290 仍远低于 AF3 置信阈值（0.7），整体预测可信度有限。

5.2 模板偏差：AF3 结果不等于结合亲和力

重要局限：PDB 9L1N 正是 WEEV E2 WT（V265）的晶体结构。 AF3 对 WT 预测有完美模板支撑，因此 WT 的 ipTM 系统性高于 V265F。 这一差距（Δ=0.110，R2轮）不能直接解读为"WT 与 VLDLR 亲和力更高"， 更可能反映模板偏差（template bias）而非真实生物学差异。

5.3 V265 不是直接接触残基

有价值的阴性发现：对全部3种受体，265位残基的接触概率均接近于零（WT 和 V265F 均如此）。 这说明 V265F 的受体切换效应很可能是间接机制—— 265位的 Val→Phe 侧链改变影响 E2 整体构象，进而调控远端结合界面， 而非265位本身直接插入受体结合口袋。 这与该位点位于 E2 疏水核心区而非暴露表面的结构逻辑一致。

5.4 PCDH10 界面不可预测

WT 和 V265F 对 PCDH10 的 ipTM 均约为0.14–0.15，无鉴别力。 原因是 PDB 中没有任何甲病毒 E2–PCDH10 复合物结构，AF3 缺乏模板，预测纯靠序列推断。 这一结果与 PCDH10 作为"新发现受体"的现实一致，不代表 E2-PCDH10 无结合。

6. 方法局限性

  • AF3 ipTM 是预测姿势的置信度，不等于结合亲和力（ΔG）
  • 模板偏差导致有晶体结构的 WT 系统性高于突变体
  • VLDLR 复合物缺乏实验结构，AF3 预测误差难以校验
  • AF3 Server 每组仅5个随机种子，采样可能不充分
  • 未考虑 E2 三聚体状态和脂双层环境对构象的影响

7. 收获与未来展望

本次探索建立了迄今 WEEV E2 + VLDLR LA3 的最高质量 AF3 预测（ipTM=0.290）， 并通过两轮迭代设计（全长→截短）系统性地测试了输入序列长度对预测质量的影响。 核心收获：

  1. AF3 在有 PDB 模板时存在系统性模板偏差，这一局限需要在解读突变体预测结果时明确说明
  2. V265 不在直接接触界面的发现，提示该残基通过变构而非直接接触机制影响受体选择
  3. 建立了可复用的分析流程（序列准备 → AF3 提交 → JSON 解析 → 可视化）
